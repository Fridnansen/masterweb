Concepto del juego

Es una variante de ajedrez para cuatro jugadores con reglas que permiten la interacción tanto de amistad como rivalidad en humano/máquina de forma bilateral (pareja) o unilateral (individual).


Proyecto

Se propone desarrollar campeonatos con juegos de mesa, la página web con torneos en línea y la aplicación para dispositivos móviles.
