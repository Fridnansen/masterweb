Masterdrez
Es una variante de ajedrez para cuatro jugadores con reglas que permiten la interacción tanto de amistad como rivalidad en pareja o individual.
Fundación Masterdrez
Fue creada con el proposito de impulsar proyectos que utilicen materiales o herramientas ludicas, para elevar el cociente intelectual del público general.
